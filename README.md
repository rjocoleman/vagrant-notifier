# Vagrant::Notifier

For Vagrant 1.5+.

Sends a notification via [TerminalNotifier](https://github.com/alloy/terminal-notifier) when `$ vagrant` is finished.

## Installation

`$ vagrant plugin install vagrant-notifier`

## Usage

It's automatic just `$ vagrant up`!
